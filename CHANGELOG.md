## v0.5.0 - 2026-01-10
### Added
* Added the stored score to the higher store API
* Added Block Type 11 (seems to be unused)
* Minefield: added the cansee bits previously unknown
* Mineral Packet: added PacketMaxWeight & PacketDecayRate, previoulsy unknown
* Wormhole: added StabilityIndex, TurnsSinceMove, DestKnown, IncludeInDisplay
* CometStrike: fully decoded event
### Changed
* Update MessageBlock Type 40 with all fields
* Salvage: added SalvageSourceFlags
* Wormholes: Fixed swapped fields: CanSeeBits is now bytes 8-9, BeenThroughBits is now bytes 10-11
* Wormholes: TargetId now properly masked with & 0x0FFF, Stability constants updated to 4-value index
* Fleet: updated the last unknown bits with discovered values
* WaypointRepeatOrders: found the bit enabling disabling repeat on waypoint orders
